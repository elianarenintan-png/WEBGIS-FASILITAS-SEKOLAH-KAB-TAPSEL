# WEBGIS-FASILITAS-SEKOLAH-KAB-TAPSEL
WebGIS ini menyajikan informasi spasial dan atribut sekolah di Kabupaten Tapanuli Selatan secara interaktif, meliputi nama sekolah, status sekolah, alamat lengkap, jumlah siswa, jumlah guru, akreditasi, tahun berdiri, koordinat lokasi (latitude dan longitude), serta tautan Google Maps
